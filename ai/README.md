# AI Components

Defines telemetry normalization, API contracts, prompt assets, and runtime configuration for AI-assisted insights.

- `normalizer/` – JSON Schemas and examples for metrics/events/reports plus redaction rules.
- `api/` – Read-only API specification and endpoint notes.
- `prompts/` – System and task prompts for summarization and anomaly detection.
- `runtime/` – Optional model runtime and environment configuration.
